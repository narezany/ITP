# ИТП — кастомный клиент для ИТД

Android-приложение для [итд](https://итд.com/) с расширенными функциями. Не является форком нативного мобильного приложения, использует веб-версию.
### Для открытия меню твиков зажмите иконку приложения и выберите "Твики и настройки".

## Функции 

### Фильтры контента

| Функция | Описание |
|---------|----------|
| **Блюр матов** | Размывает текст постов с нецензурной лексикой. Нажми на пост: размытие снимается |
| **Экономия трафика** | Полностью скрывает все изображения и видео в ленте |
| **Скачка фото** | Кнопка ⬇ в правом верхнем углу каждого фото в посте. Сохраняет в `Pictures/ITP/` |
| **Скрытие эмодзи-кланов** | Введи эмодзи кланов — посты от этих кланов полностью скрываются |

### Переводчик

Встроенный перевод постов через Google Translate:

- **Не переводить с языков** — список кодов языков через запятую (например `ru`)
- **Переводить на язык** — целевой язык (например `ru`)
- **Авто-режим** — все посты переводятся сразу
- **Ручной режим** — рядом с кнопками ❤ 💬 🔄 появляется кнопка перевода. Нажми → перевод загрузится → текст заменится. Нажми ещё раз → вернётся оригинал

### Оформление и интерфейс

| Функция | Описание |
|---------|----------|
| **Язык приложения** | Выбор предпочитаемого языка (Системный, Английский, Русский, Китайский, Испанский) |
| **Material You** | Адаптирует цветовую палитру сайта под системные Material You цвета (Android 12+). Поддержка светлой и тёмной темы |
| **ПК версия** | Сайт как на компьютере, поддержка горизонтального экрана |

### Безопасность и система

| Функция | Описание |
|---------|----------|
| **PIN-код** | Запрашивает 4-значный PIN при каждом холодном запуске приложения |
| **Аналитика** | Отключаемый сбор базовой анонимной статистики использования твиков |

### Прочее

- **Pull-to-refresh** — потяни вниз вверху страницы для обновления
- **Ярлык** — при долгом нажатии на иконку доступен быстрый переход в Твики

## 🛠 Сборка

### Требования

- Android SDK (API 34)
- JDK 11+
- Gradle

### Команды

```bash
# Debug APK
./gradlew assembleDebug

# Release APK (нужен keystore)
./gradlew assembleRelease
```
